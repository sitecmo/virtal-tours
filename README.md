# CMO Virtual Tours

Servidor Node para mapear tours e gerar `tours-data.js`/`tours-map.json` para uso estático.

## Como rodar (curto)

- `npm install`
- `npm run map` para gerar dados
- Opcional local: `npm start` (porta 3030) para servir estático e API

## GitHub Pages

- Após `npm run map`, suba `index.html`, `tours-data.js`, `tours-map.json` e `src/`.
- Configure Pages (branch + root) e acesse `https://<usuario>.github.io/<repo>/`.
